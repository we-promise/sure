# India Tax Workbook Import Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Settings -> Tax Calculations feature that uploads a normalized India GST/TDS `.xlsx` workbook, validates it, parses it into structured family-scoped records, and lists/searches/report totals inside Sure.

**Architecture:** Add a dedicated tax-workbook domain parallel to Statement Vault and Family Exports: Active Storage keeps the source workbook, service objects parse/generate XLSX files, and normalized Active Record models store GST/TDS rows. Controllers stay thin and render settings-layout pages for upload history, validation details, and searchable parsed-record reports.

**Tech Stack:** Rails 7.2, Active Storage, Minitest fixtures, `roo` for XLSX parsing, `caxlsx` for XLSX template generation, Pagy for listings, existing DS/settings view helpers.

---

## Research Inputs

- Context7 resolved `/roo-rb/roo`: `Roo::Spreadsheet.open(path, extension: :xlsx)` supports sheet listing, row access, and hash-style parsing.
- Context7 resolved `/caxlsx/caxlsx`: `Axlsx::Package` can create worksheets with `add_worksheet`, `add_row`, styles, `to_stream`, and `validate`.
- Context7 Rails 7.2 docs confirm `has_one_attached`, `rails_blob_path(..., disposition: "attachment")`, `purge_later`, and `file_fixture_upload`.
- Context7 Pagy docs confirm controller use with `pagy(:offset, relation, limit:)`; this repo uses the older-compatible local shape `pagy(relation, limit:)`, so follow the repo.
- Firecrawl/RubyGems lookup: `roo` current line is `3.0.0` and requires Ruby `>= 3.1.0`; this repo declares Ruby `3.4.7`, so use current `roo`.
- Firecrawl/RubyGems lookup: `caxlsx` current line is `4.5.0` and supports Ruby `>= 2.6`; use current `caxlsx`.
- Firecrawl lookup found `simple_xlsx_reader` as a streaming alternative. Keep it out of the first implementation because the workbook template is bounded and `roo` has better ergonomic sheet/header APIs.
- Firecrawl lookup found `axlsx_rails` last released in 2019. Do not add it; generate template bytes from a plain service and return them with `send_data`.

```ruby
class TaxWorkbookImport < ApplicationRecord
  XLSX_CONTENT_TYPE = "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"
  MAX_FILE_SIZE = 10.megabytes

  belongs_to :family
  belongs_to :uploaded_by, class_name: "User"

  has_one_attached :source_file, dependent: :purge_later
  has_many :gst_outward_lines, dependent: :destroy
  has_many :gst3b_summaries, dependent: :destroy
  has_many :gst_hsn_summaries, dependent: :destroy
  has_many :tds_deductions, dependent: :destroy
  has_many :tds_challans, dependent: :destroy

  enum :status, {
    pending: "pending",
    validated: "validated",
    importing: "importing",
    complete: "complete",
    failed: "failed"
  }, default: :pending, validate: true

  scope :ordered, -> { order(created_at: :desc) }

  validates :filename, :content_type, :byte_size, :checksum, :template_version, presence: true
  validates :byte_size, numericality: { greater_than: 0, less_than_or_equal_to: MAX_FILE_SIZE }
  validates :content_type, inclusion: { in: [ XLSX_CONTENT_TYPE ] }

  def gst_tax_total
    gst_outward_lines.sum("igst + cgst + sgst_ugst + cess")
  end

  def gst_taxable_total
    gst_outward_lines.sum(:taxable_value)
  end

  def tds_total
    tds_deductions.sum(:tds_amount)
  end
end
```

```ruby
module TaxWorkbook
  class ValueParser
    TRUE_VALUES = %w[true t yes y 1].freeze
    FALSE_VALUES = %w[false f no n 0].freeze
    GSTIN_PATTERN = /\A[0-9]{2}[A-Z0-9]{13}\z/
    TAN_PATTERN = /\A[A-Z]{4}[0-9]{5}[A-Z]\z/
    PAN_PATTERN = /\A[A-Z]{5}[0-9]{4}[A-Z]\z/

    def decimal(value)
      return BigDecimal("0") if value.blank?

      BigDecimal(value.to_s.delete(","))
    rescue ArgumentError
      raise ArgumentError, "must be a decimal number"
    end

    def boolean(value)
      normalized = value.to_s.strip.downcase
      return true if TRUE_VALUES.include?(normalized)
      return false if FALSE_VALUES.include?(normalized)

      raise ArgumentError, "must be true or false"
    end

    def date(value)
      return value.to_date if value.respond_to?(:to_date)

      Date.parse(value.to_s)
    rescue ArgumentError
      raise ArgumentError, "must be a date"
    end

    def month(value)
      parsed = value.to_s.match?(/\A\d{4}-\d{2}\z/) ? Date.strptime("#{value}-01", "%Y-%m-%d") : date(value)
      parsed.beginning_of_month
    rescue ArgumentError
      raise ArgumentError, "must be a month like YYYY-MM"
    end

    def quarter(value)
      normalized = value.to_s.strip.upcase
      return normalized if normalized.in?(%w[Q1 Q2 Q3 Q4])

      raise ArgumentError, "must be Q1, Q2, Q3, or Q4"
    end

    def gstin(value)
      normalized = compact_identifier(value)
      raise ArgumentError, "must be a 15-character GSTIN" unless normalized.match?(GSTIN_PATTERN)

      normalized
    end

    def tan(value)
      normalized = compact_identifier(value)
      raise ArgumentError, "must be a 10-character TAN" unless normalized.match?(TAN_PATTERN)

      normalized
    end

    def pan(value)
      normalized = compact_identifier(value)
      raise ArgumentError, "must be a 10-character PAN" unless normalized.match?(PAN_PATTERN)

      normalized
    end

    private

      def compact_identifier(value)
        value.to_s.strip.upcase.gsub(/\s+/, "")
      end
  end
end
```

```ruby
module TaxWorkbook
  class SheetReader
    def initialize(path)
      @workbook = Roo::Spreadsheet.open(path, extension: :xlsx)
    end

    def sheet_names
      @workbook.sheets.map { |sheet| normalize_name(sheet) }
    end

    def rows(sheet_name)
      sheet = @workbook.sheet(find_sheet_name(sheet_name))
      header = sheet.row(1).map { |cell| normalize_name(cell) }

      ((sheet.first_row || 2) + 1..(sheet.last_row || 1)).filter_map do |row_number|
        values = sheet.row(row_number)
        next if values.all?(&:blank?)

        row = header.zip(values).to_h
        row["source_row_number"] = row_number
        row
      end
    end

    private

      def find_sheet_name(sheet_name)
        @workbook.sheets.find { |candidate| normalize_name(candidate) == normalize_name(sheet_name) } ||
          raise(KeyError, "Missing sheet #{sheet_name}")
      end

      def normalize_name(value)
        value.to_s.strip.downcase.tr(" -", "__").gsub(/_+/, "_")
      end
  end
end
```

```ruby
module TaxWorkbook
  class Parser
    BOOLEAN_COLUMNS = %w[is_reverse_charge is_export is_ecommerce_tcs is_credit_note is_debit_note].freeze

    def initialize(family:, uploaded_by:, file:)
      @family = family
      @uploaded_by = uploaded_by
      @file = file
      @value_parser = ValueParser.new
      @errors = []
    end

    def call
      content = read_file!
      import = build_import(content)
      import.source_file.attach(io: StringIO.new(content), filename: import.filename, content_type: import.content_type)
      import.save!

      tempfile = write_tempfile(content)
      reader = SheetReader.new(tempfile.path)
      validate_required_sheets!(reader)
      meta = parse_meta(reader.rows("meta").first || {})
      import.update!(meta.merge(status: "importing"))

      ActiveRecord::Base.transaction do
        create_gst_outward_lines!(import, reader.rows("gst_outward_lines"))
        create_gst3b_summaries!(import, reader.rows("gst_3b_summary"))
        create_gst_hsn_summaries!(import, reader.rows("gst_hsn_summary"))
        challans = create_tds_challans!(import, reader.rows("tds_challans"))
        create_tds_deductions!(import, reader.rows("tds_deductions"), challans)
        import.update!(status: "complete", row_counts: row_counts(import))
      end

      ImportResult.new(true, import, [])
    rescue StandardError => e
      @errors << { "message" => e.message } if @errors.empty?
      import&.update!(status: "failed", validation_errors: @errors)
      ImportResult.new(false, import, @errors)
    ensure
      tempfile&.close!
    end

    private

      def read_file!
        raise ArgumentError, "Choose an XLSX file" unless @file&.respond_to?(:read)
        raise ArgumentError, "File is too large" if @file.size.to_i > TaxWorkbookImport::MAX_FILE_SIZE

        content = @file.read
        @file.rewind if @file.respond_to?(:rewind)
        raise ArgumentError, "File is empty" if content.blank?

        content
      end

      def build_import(content)
        @family.tax_workbook_imports.build(
          uploaded_by: @uploaded_by,
          filename: @file.original_filename.to_s,
          content_type: TaxWorkbookImport::XLSX_CONTENT_TYPE,
          byte_size: content.bytesize,
          checksum: Digest::SHA256.hexdigest(content),
          template_version: Template::VERSION,
          status: "pending"
        )
      end

      def write_tempfile(content)
        tempfile = Tempfile.new([ "tax-workbook", ".xlsx" ])
        tempfile.binmode
        tempfile.write(content)
        tempfile.rewind
        tempfile
      end

      def validate_required_sheets!(reader)
        missing = Template::SHEET_NAMES - reader.sheet_names
        return if missing.empty?

        raise ArgumentError, "Missing workbook sheets: #{missing.join(', ')}"
      end

      def parse_meta(row)

          entity_name: row.fetch("entity_name").to_s,
          gstin: @value_parser.gstin(row.fetch("gstin")),
          tan: @value_parser.tan(row.fetch("tan")),
          fy: row.fetch("fy").to_s,
          tax_period_month: @value_parser.month(row.fetch("tax_period_month")),
          tax_period_quarter: @value_parser.quarter(row.fetch("tax_period_quarter"))
        }
      rescue KeyError, ArgumentError => e
        row_error("meta", row["source_row_number"] || 2, "meta", nil, e.message)
        raise
      end

            source_row_number: row.fetch("source_row_number"),
            tax_period_month: import.tax_period_month,
            gstin: import.gstin,
            gstr1_table_code: row.fetch("gstr1_table_code").to_s,
            invoice_no: row.fetch("invoice_no").to_s,
            invoice_date: @value_parser.date(row.fetch("invoice_date")),
            recipient_gstin_or_uin: row["recipient_gstin_or_uin"].to_s.presence,
            place_of_supply_state: row["place_of_supply_state"].to_s.presence,
            hsn_code: row["hsn_code"].to_s.presence,
            rate_pct: @value_parser.decimal(row["rate_pct"]),
            taxable_value: @value_parser.decimal(row["taxable_value"]),
            igst: @value_parser.decimal(row["igst"]),
            cgst: @value_parser.decimal(row["cgst"]),
            sgst_ugst: @value_parser.decimal(row["sgst_ugst"]),
            cess: @value_parser.decimal(row["cess"]),
            is_reverse_charge: @value_parser.boolean(row["is_reverse_charge"]),
            is_export: @value_parser.boolean(row["is_export"]),
            is_ecommerce_tcs: @value_parser.boolean(row["is_ecommerce_tcs"]),
            is_credit_note: @value_parser.boolean(row["is_credit_note"]),
            is_debit_note: @value_parser.boolean(row["is_debit_note"])
```

```ruby
class TaxWorkbookImportsController < ApplicationController
  before_action :require_admin
  before_action :set_import, only: %i[show download destroy]

  def index
    @pagy, @imports = pagy(Current.family.tax_workbook_imports.with_attached_source_file.ordered, limit: safe_per_page)
    @breadcrumbs = [ [ t("breadcrumbs.home"), root_path ], [ t("breadcrumbs.tax_workbook_imports"), tax_workbook_imports_path ] ]
    render layout: "settings"
  end

  def show
    @breadcrumbs = [
      [ t("breadcrumbs.home"), root_path ],
      [ t("breadcrumbs.tax_workbook_imports"), tax_workbook_imports_path ],
      [ @import.filename, nil ]
    ]
    render layout: "settings"
  end

  def create
    file = tax_workbook_import_params[:file]
    result = TaxWorkbook::Parser.new(family: Current.family, uploaded_by: Current.user, file: file).call

    if result.success?
      redirect_to tax_workbook_import_path(result.import), notice: t(".success")
    else
      redirect_to tax_workbook_import_path(result.import), alert: t(".failed")
    end
  rescue ArgumentError => e
    redirect_to tax_workbook_imports_path, alert: e.message
  end

  def template
    send_data TaxWorkbook::TemplateGenerator.new.call,
              filename: "sure_india_tax_workbook_template_v#{TaxWorkbook::Template::VERSION}.xlsx",
              type: TaxWorkbookImport::XLSX_CONTENT_TYPE,
              disposition: "attachment"
  end

  def download
    redirect_to rails_blob_path(@import.source_file, disposition: "attachment")
  end

  def destroy
    @import.destroy
    redirect_to tax_workbook_imports_path, notice: t(".success")
  end

  private

    def set_import
      @import = Current.family.tax_workbook_imports.with_attached_source_file.find(params[:id])
    end

    def tax_workbook_import_params
      params.fetch(:tax_workbook_import, ActionController::Parameters.new).permit(:file)
    end

    def require_admin
      redirect_to root_path, alert: t("accounts.not_authorized") unless Current.user.admin?
    end
end
```

```ruby
      relation = Current.family.gst_outward_lines
      search = @q[:search].to_s.strip
      return relation if search.blank?

      relation.where("invoice_no ILIKE :q OR gstin ILIKE :q OR recipient_gstin_or_uin ILIKE :q", q: "%#{search}%")
    end

    def require_admin
```
